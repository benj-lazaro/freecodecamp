The Classic Mac OS

The operating system of the Macintosh personal computer family from 1984 to 2001

Version Highlights

  • System 1 (January 1984): Initial release
  • System 2 (January 1985): Finder 5.0 update, Added support for AppleTalk, Hard Disk 20, LaserWriter and introduced multiple folders, "Shut Down" command & screenshots
  • System 3 (January 1986): Hierarchical File System (HFS) implementation, Added support for 800K startup drives, SCSI & Trash "bulging"
  • System 4 (January 1987): introduced AppleShare. LaserWriter driver update
  • System 5 (October 1987): Cooperative multitasking, LaserWriter driver update, New Apple HD SC setup
  • System 6 (April 1998): 68030 processor and 1.44Mb SuperDrive support, Consolidated release that featured a complete & stable OS, Macintosh Portble support
  • System 7 (May 13, 1991): Changed name to Mac OS (7.6 release), UI overhaul, Imporved Stability, Extension Manager, Multitasking and Virtual Memory Manager support
  • System 8 (July 26, 1997): Dropped support for Macintosh II & 68030-based Macs, Support for Hierarchical File System+ (HFS+) and PowerPC processor, Required a 68040 processor to install
  • System 9 (October 12, 1999): Final release of the Classic Mac OS, improved Mac OS X compatibility
